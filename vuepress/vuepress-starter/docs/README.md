在数字内容爆炸的今天，时间是您最宝贵的资产。通用网页倍速控制器不仅仅是一个工具，更是您高效获取信息、掌控娱乐节奏的强力助手。

### 🚀 核心能力

::: details 点击展开核心技术细节

- **全域覆盖能力**：打破平台壁垒，从 Bilibili 到 YouTube，从网课平台到 4399 小游戏，一个脚本，全网通用。
- **无级变速控制**：突破传统播放器 2.0x 限制，支持 0.1x 至 16.0x 任意精度的调整。
- **游戏辅助加速**：专为 Unity/Cocos/LayaAir 网页游戏优化，轻松跳过繁琐剧情，快速跑图，体验飞一般的感觉。 :::

### 💡 深度技术原理

不同于传统插件仅修改 DOM 属性，本核心引擎深入浏览器运行时（Runtime）：

1. **时间流代理 (Time Stream Proxy)**：劫持 `Date` 与 `Performance` API，重塑底层时间坐标系。
2. **渲染重定向 (Render Redirection)**：接管 `requestAnimationFrame`，让渲染循环跟随您的节奏律动。
3. **多态自适应 (Polymorphic Adaptation)**：自动识别 DOM 媒体与 Canvas 画布，智能匹配最佳加速策略。
